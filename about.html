About

The [OFFICIAL] Creator School

The [OFFICIAL] Creator School Discord server was created on the 17th February 2018 by Veztix. He enlisted Chaoz18 and ChiefJack to help setup the server and kick start it. The server was relatively successful from the beginning, however it hasn't been a smooth ride. The server has seen numerous ownership changes, including Veztix, Chaoz18, Chlorines, and more recently, StrikeKing22 and Sequex, all leaving or resigning.

However, the server is now doing very well. The server now has its own Discord bot to vastly improve how the server runs, and its own website to attract further members (where you're reading this)!

The server boasts 13 detailed and concise Classes to help YOU improve your GD creating skills. With over 30 Teachers and 10 Tutors, help will always be at hand in this very active server!

In a recent user survey, 100% of people said they would recommend the server to other creators, and over 90% rated the server 7/10 or higher!
